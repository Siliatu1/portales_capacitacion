La Escuela del Café es un módulo de inscripciones para la capacitación de colaboradoras de heladerías y restaurantes, donde se gestiona mensualmente la programación de inscripciones y las instructoras pueden llevar el control de asistencia de cada participante.


Evaluación Toderas es un módulo de inscripción para colaboradoras de restaurantes, diseñado para gestionar y realizar el proceso de evaluación de su cargo.
